# Lantern Bloom 

This is a simple restaurant website built with HTML.

I created this project to practice structuring a webpage and building something with a clean visual idea behind it. The project helped me get more comfortable with basic web development and creating a site that feels more like a real business page.

## Concepts Practiced

- HTML
- Webpage structure
- Basic layout
- Front-end design
- Restaurant website concept

## What I Learned

This project helped me understand how HTML can be used to organize content and create the foundation for a simple website.
